# SO_Lista02.02
## Matar Processos

Este desafio consiste em criar uma aplicação, em Java, que mate o processo ao entrar com seu nome ou PID.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A plicação deve identificar o Sistema Operacional para realizar o processo para Linux e Windows;
3. Para matar os processos, deve-se exibir a tabela de processos.
```
